💾 Lưu & Xuất dữ liệu (Excel)

File Excel xuất ra vừa là báo cáo đầy đủ, vừa có thể nạp lại để tiếp tục nhập liệu lần sau.

📊

Xuất Excel Phiếu Nông Dân

Xuất dữ liệu ra file .xlsx (Microsoft Excel). File gồm nhiều sheet báo cáo và tự động lưu dữ liệu thô để nạp lại — dùng 1 file cho tất cả.
📂

Nạp dữ liệu từ file Excel đã lưu

Chọn file .xlsx đã lưu bằng nút "Lưu dữ liệu ra Excel" ở trên. Toàn bộ cấu hình và phiếu nông dân sẽ được khôi phục.
